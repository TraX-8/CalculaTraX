# CalculaTraX
## Calculatrice (par [*TraX Software*](http://trax8.fr.nf/trax-software))
---
> Comment installer l'application via la dépôt **GitHub** (pour Linux) ?
1. Télécharger l'archive compressée du code (fichier `.tar.gz`)
2. Ouvrir l'explorateur de fichiers à l'endroit où vous avez téléchargé l'archive.
3. Décompresser l'archive.
4. Entrer dans le dossier qui a été créé, puis faire un clic droit, et sélectionner "Ouvrir un terminal".
5. Entrez `chmod +x script.sh`, puis `sudo bash script.sh`.
6. Le script d'installation étant lancé, il ne vous reste plus qu'à patienter jusqu'à ce que l'application soit installée.
7. Quand le message de confirmation d'installation réussie s'affiche, l'application est installée. Pour que l'application s'affiche dans le menu principal, un redémarrage sera peut-être requis.
8. **Profitez de l'application !**
